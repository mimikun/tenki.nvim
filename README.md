# tenki.nvim

**SORRY JAPANESE ONLY**

A neovim plugin to display tenki(weather) in Japan

## Features

- Show tenki

### Commands

- `ShowFukui` to Show Fukui Tenki.
- `ShowNagoya` to Show Nagoya Tenki.
- `ShowTokyo` to Show Tokyo Tenki.

## Setup

### System requirements

- Neovim nightly or HEAD
  - Testing only on nightly and HEAD

### Install

#### lazy.nvim

```lua
{ "mimikun/tenki.nvim" }
```
